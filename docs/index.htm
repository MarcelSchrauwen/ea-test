Welkom

Deze statische html publicatie bevat een doorklikbare versie van de enterprise architect repositoy EA-GIS, die zelf op een beveiligde plek staat.


Deze omgeving is bedoeld als architectuurhulpmiddel om tot een integraal overzicht te komen van het gezondheidsinformatiestelsel: zowel de huidige situatie met beperkte samenhang, als de gewenste toekomstige situatie met meer samenhang.


Deze omgeving is niet bedoeld om primair te gebruiken voor brede communicatie. Hiervoor zal een omgeving worden ingericht.


De opmaak van deze publicatie zal overigens steeds een beetje beter gemaakt worden, zodat deze steeds een beetje toegankelijker wordt


De huidige focus is gericht op het komen tot een geaccepteerde lijst van applicatiefuncties.
